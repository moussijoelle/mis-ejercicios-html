Practicar arrays

Crea un array de lo que gustes (citas, animales, flores) en JS.

Hacer un programa con dos botones: uno para aumentar la posición a mostrar y otro para disminuirla.

Una vez que la posición aumente o disminuya, mostrar el contenido del array en esa posición.

Pistas: usar if para que al disminuir no baje de 0 y al aumentar no salga del límite superior. La variable que guarda la posición a mostrar se define fuera de las funciones.
