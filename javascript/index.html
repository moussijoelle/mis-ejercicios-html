Tipos de Datos en JavaScript

Primitivos: Se accede directamente al valor. Para saber el tipo de datos de una variable se usa typeof

  • string: Cadena de Texto. Se convierte a número con la función Number.
  • number: Valor numérico
  • bigint: Números enteros mayores a 9007199254740991
    (253 - 1). Se representan añadiendo una n al final del número.
  • boolean: Valor lógico Verdadero/Falso. (ver Falsy y Truthy)
  • null: La variable tiene como valor, vacío.
  • undefined: La variable ha sido declarada pero no tiene valor.
  • NaN: Not a Number (no es un número)

Compuestos: Se accede a la referencia del valor. La variable no guarda los datos, guarda la "ubicación" en la memoria donde están esos datos. Si dos variables apuntan a la misma dirección y una cambia algo, la otra también lo verá cambiado. Para saber el tipo de objeto que se está trabajando se utiliza instanceof.

  • object: {...} Es una colección de clave: valor.
  • array: [...] Guardan listas de valores los cuales referenciamos por un índice numérico.
  • function(): {...} Guardan un bloque de código para ejecutarlo al ser invocado.
  • Class: {...} Plantillas que permiten crear muchos objetos con la misma estructura y comportamiento
