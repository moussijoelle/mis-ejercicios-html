Test — 15 preguntas

Responde mentalmente y pulsa cada pregunta para desplegar la respuesta (<details>).

  1. ¿Qué debe introducir el usuario en este ejercicio?

    Un string (palabra o frase de texto), no un número.

  2. Si el usuario escribe «hola», ¿qué debe mostrar el programa?

    aloh (las letras en orden inverso).

  3. ¿Qué estructura de control exige el enunciado del ejercicio?

    Un bucle for para recorrer el string de la última letra a la primera.

  4. ¿Por qué el for empieza en texto.length - 1?

    Porque los índices empiezan en 0; la última letra está en la posición longitud − 1.

  5. ¿Qué hace invertido += texto[i] en cada vuelta del bucle?

    Añade la letra de la posición i al final de la variable invertido.

  6. ¿Para qué sirve id="formTexto"?

    Para que JavaScript seleccione el formulario con getElementById('formTexto').

  7. ¿Qué hace evento.preventDefault()?

    Evita que el navegador recargue la página al enviar el formulario.

  8. ¿Para qué sirve el atributo for="texto" en el label?

    Asocia la etiqueta al input con id="texto"; al pulsar el label, el cursor va al campo.

  9. ¿Qué archivo contiene la lógica de invertir el texto?

    js/app.js, enlazado desde el HTML con <script src="js/app.js">.

  10. ¿Dónde se muestra el resultado en pantalla?

    En el div con id="resultado", rellenado por zonaResultado.innerHTML.

  11. ¿Qué significa src="img/cccircular.svg"?

    Ruta del archivo de imagen de fondo (origen = source).

  12. ¿Por qué i-- en el for?

    Para ir restando el índice y leer el string de derecha a izquierda.

  13. ¿Qué pasa si el usuario envía el formulario vacío?

    El navegador bloquea el envío y muestra un aviso por el atributo required.

  14. ¿Qué devuelve stringAlReves('') (string vacío)?

    Un string vacío ''; el bucle no se ejecuta ninguna vez.

  15. ¿Qué color rosa usa el botón «Mostrar al revés»?

    bg-[#cb839a] (y al hover #b86d88), en armonía con el fondo SVG.
