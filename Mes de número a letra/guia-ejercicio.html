Guía del ejercicio «Mes de número a letra»

En qué consistía HTML línea a línea JavaScript línea a línea 15 preguntas tipo ABC

En qué consistía el ejercicio

El objetivo era construir una pequeña aplicación web que relacione un número de mes (1 a 12) con el nombre del mes en español (Enero, Febrero, etc.).

Para eso se usa un array en JavaScript donde la posición 0 guarda null a propósito: así el índice del array coincide con el número del mes (el mes 1 está en la posición 1, no en la 0).

La página pide al usuario un número entre 1 y 12 (campo numérico y botón «Mostrar»), valida la entrada y muestra el mensaje pedido en el enunciado, por ejemplo: «El mes 1 se corresponde con Enero.»

15 preguntas tipo ABC (para tu quiz)

Haz el quiz leyendo solo la pregunta y las opciones A, B y C. No abras el bloque «Respuesta» hasta haber respondido: está hecho con <details> para ocultar la solución.

1. ¿Para qué se guarda null en la primera posición del array mesesDelAno?

  1. Para indicar que no hay mes en diciembre
  2. Para que el índice 1 del array corresponda al mes 1 (enero)
  3. Porque JavaScript obliga a empezar siempre con null
Respuesta (abrir después del quiz)

B. La posición 0 no se usa como mes; así mesesDelAno[1] es Enero, mesesDelAno[2] es Febrero, etc.

2. ¿Qué método del documento usa el código para enlazar el input con JavaScript?

  1. document.querySelectorAll
  2. document.getElementById
  3. document.createElement
Respuesta (abrir después del quiz)

B. Se usa getElementById("numeroMes") (y lo mismo para el botón y el mensaje).

3. ¿Qué atributo HTML conecta la etiqueta «Mes (1–12):» con el campo de número?

  1. name en el input
  2. for en el label igual al id del input
  3. class con el mismo valor en ambos
Respuesta (abrir después del quiz)

B. for="numeroMes" en el label apunta a id="numeroMes" en el input (accesibilidad y clic en la etiqueta).

4. ¿Qué hace el atributo defer en <script src="app.js" defer>?

  1. Elimina todos los errores del script
  2. Retarda la ejecución hasta después de analizar el HTML del documento
  3. Impide que el script se ejecute nunca
Respuesta (abrir después del quiz)

B. Con defer el navegador descarga el script en paralelo pero lo ejecuta cuando el HTML está parseado, antes de DOMContentLoaded.

5. ¿Por qué se usa Number.isInteger(numeroMes) tras Number(texto)?

  1. Para convertir el número a cadena
  2. Para rechazar valores como 3.5 o NaN que no son meses enteros
  3. Para redondear automáticamente al mes más cercano
Respuesta (abrir después del quiz)

B. Los meses del enunciado son enteros del 1 al 12; un decimal no es válido para este ejercicio.

6. ¿Qué propiedad del párrafo se modifica para mostrar el resultado en pantalla?

  1. innerHTML
  2. textContent
  3. value
Respuesta (abrir después del quiz)

B. Se asigna a parrafoMensaje.textContent; es adecuado para texto plano sin etiquetas HTML.

7. ¿Qué etiqueta HTML agrupa el contenido principal de la página de forma semántica?

  1. <section>
  2. <main>
  3. <article>
Respuesta (abrir después del quiz)

B. El contenido principal del ejercicio está dentro de <main>.

8. ¿Qué evento se escucha en el botón para lanzar mostrarNombreDelMes?

  1. submit
  2. click
  3. change
Respuesta (abrir después del quiz)

B. botonMostrar.addEventListener("click", mostrarNombreDelMes).

9. Si el usuario pulsa Enter dentro del input, ¿qué hace el código?

  1. Borra el contenido del input
  2. Llama también a mostrarNombreDelMes
  3. Recarga la página automáticamente
Respuesta (abrir después del quiz)

B. En keydown, si evento.key === "Enter", se ejecuta la misma función que al pulsar Mostrar.

10. ¿Cuántos elementos con nombre de mes hay en el array mesesDelAno (sin contar null)?

  1. 10
  2. 11
  3. 12
Respuesta (abrir después del quiz)

C. Hay 12 cadenas desde Enero hasta Diciembre; más null en el índice 0.

11. ¿Para qué sirve aria-live="polite" en el párrafo del mensaje?

  1. Para que el texto aparezca en negrita
  2. Para avisar de forma no intrusiva a tecnologías de accesibilidad cuando cambia el contenido
  3. Para desactivar el lector de pantalla
Respuesta (abrir después del quiz)

B. Indica que las actualizaciones del mensaje pueden anunciarse al usuario sin interrumpir de forma agresiva.

12. ¿Qué devuelve mesesDelAno[6] en tu código?

  1. Mayo
  2. Junio
  3. Julio
Respuesta (abrir después del quiz)

B. Índice 6 es la sexta cadena después de null: Enero=1, …, Junio=6.

13. ¿Qué atributos del input limitan el rango en la interfaz del navegador (según tu HTML)?

  1. min y max
  2. start y end
  3. low y high
Respuesta (abrir después del quiz)

A. min="1" y max="12" en el input tipo number.

14. En el mensaje de éxito, ¿cómo se insertan el número y el nombre del mes en la cadena?

  1. Con el operador + solo entre cadenas fijas
  2. Con un template literal usando ${…}
  3. Con la función alert()
Respuesta (abrir después del quiz)

B. Se usa backtick (`) y expresiones ${numeroMes} y ${nombreMes}.

15. ¿Qué hace return dentro de los if de validación en mostrarNombreDelMes?

  1. Devuelve un valor al servidor
  2. Detiene la función en ese punto sin ejecutar el resto
  3. Repite el bucle desde el principio
Respuesta (abrir después del quiz)

B. Tras mostrar el error, return evita seguir y buscar el mes en el array con datos inválidos.
